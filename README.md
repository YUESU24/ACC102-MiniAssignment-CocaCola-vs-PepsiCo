# Profitability Analysis: Coca-Cola vs PepsiCo 
## 1. Problem & User
This project compares the profitability of Coca-Cola and PepsiCo to identify which company performs better. 
The target users are investors and financial analysts.
## 2. Data 
The dataset is obtained from WRDS and includes financial data from 2015 to 2025. 
Key fields include net income, total assets, shareholders’ equity, sales, and operating income. 
The data were accessed in April 2026.
## 3. Methods 
The analysis is conducted using Python. 
Key steps include data loading, cleaning, and calculation of profitability ratios such as ROA, ROE, profit margin, and operating margin. 
Line charts are used for visualization.
## 4. Key Findings 
- Coca-Cola maintains stronger profit and operating margins 
- PepsiCo exhibits more fluctuations in profitability 
- Overall, Coca-Cola demonstrates better financial performance 
## 5. How to run 
Download the dataset and place it in the same folder as the notebook. 
Run all cells in the notebook to reproduce the results.

## 7. Limitations & next steps
This analysis has several limitations. First, it only examines four financial ratios and ignores external factors like market conditions and firm-specific events that may affect performance. Second, it only involves basic calculations and visualizations, without statistical tests, regression, or forecasting models. Third, it focuses on historical trends but does not explain the underlying reasons for performance.

To improve this work, external factors such as macroeconomic and industry trends could be incorporated for a more comprehensive analysis. Advanced methods like regression or forecasting models could be added to improve accuracy. Further investigation into the drivers of financial performance would also strengthen analytical depth.
